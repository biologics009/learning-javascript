# Scope
*The region in which a defined variable or function can be accessed is called its scope.*

* If a variable or function is defined in a scope it can be accessed from the child scopes within that scope.
* They cannot be accessed outside their scope.
* Outside the scope the variable and functions are undefined.

## Types of scope

### 1. Function Scoped
Scoped within a function

### 2. Block Scoped
Scoped within {}
